# 小红书表情功能 - 快速开始 🚀

## 🎯 两种使用方式

### 方式 1：手动添加表情（推荐新手）

**适合场景**：精确控制表情位置和数量

**使用步骤**：

1. **改写完成后，点击"添加小红书表情"按钮**
   ```
   ┌─────────────────────────────────┐
   │ ✓ 复刻完成                      │
   │ 新标题、新正文...               │
   │ [😊 添加小红书表情] [保存] [发布]│
   └─────────────────────────────────┘
   ```

2. **进入表情编辑界面**
   - 标题和正文都支持添加表情
   - 点击"🥔 小红书表情"按钮
   - 选择你喜欢的表情

3. **弹出表情选择器（像手机 app 一样）**
   ```
   ┌─────────────────────────────┐
   │ 🥔 小红书官方表情        ✕ │
   ├─────────────────────────────┤
   │ 搜索表情...                 │
   ├─────────────────────────────┤
   │ 🔥热门 😊表情 👋手势       │
   ├─────────────────────────────┤
   │ 萌萌哒  暗中观察  笑哭  比心│
   │ 点赞    吃瓜    冲鸭  加油  │
   │ ...                         │
   └─────────────────────────────┘
   ```

4. **点击表情，自动插入到光标位置**
   - 实时预览效果
   - 显示表情统计
   - 查看已添加的表情

5. **点击"保存表情"**
   - 内容保存到历史版本
   - 系统自动学习你的表情偏好
   - 下次 AI 改写会更符合你的风格

**效果展示**：

改写前：
```
这款面膜真的很好用，推荐大家试试。
```

添加表情后：
```
这款面膜真的很好用[笑哭R]推荐大家试试[拿走R]
```

在小红书 app 中：
```
这款面膜真的很好用😂推荐大家试试📌
（表情会自动渲染为图片）
```

---

### 方式 2：AI 自动添加表情（效率优先）

**适合场景**：快速生成内容，让 AI 自动配表情

**使用步骤**：

1. **直接使用改写功能**
   - 输入原文
   - 点击"一键改写全部"或"改写正文"
   - AI 会自动在合适的位置添加表情

2. **AI 添加规则**：
   - 开头：吸引眼球（[派对R] [炸裂R]）
   - 重点：强调信息（[笑哭R] [赞R] [萌萌哒R]）
   - 结尾：行动号召（[拿走R] [比心R] [冲鸭R]）
   - 频率：全文 5-8 个，每段 1-2 个

3. **AI 学习你的风格**（重要！）
   - 使用"手动添加表情"功能几次后
   - AI 会学习你常用的表情
   - 后续自动添加时会优先使用你喜欢的表情

**AI 生成示例**：

输入：
```
这个小众景点人少景美还免费，周末约起来！
```

AI 输出：
```
这个小众景点人少景美还免费[派对R]周末约起来[冲鸭R]
```

## 💡 使用技巧

### 1. 表情数量建议

| 位置 | 推荐数量 | 举例 |
|-----|---------|------|
| 标题 | 0-2 个 | `这个好物太棒了[笑哭R]` |
| 短正文（<200字） | 3-5 个 | 开头1个 + 中间2个 + 结尾1个 |
| 长正文（200-500字） | 5-8 个 | 每段1-2个 |

### 2. 表情位置建议

**✅ 好的位置**：
- 句子结尾：`真的太好用了[笑哭R]`
- 关键词后：`超级推荐[赞R]`
- 段落开头：`[派对R]姐妹们听我说`
- 行动号召：`快去试试[冲鸭R]`

**❌ 避免的位置**：
- 句子中间：`这个[笑哭R]好物很不错` ❌
- 堆砌：`[火R][火R][火R]` ❌
- 无关内容：`今天下雨了[萌萌哒R]` ❌

### 3. 表情选择建议

**场景化推荐**：

| 内容类型 | 推荐表情 |
|---------|---------|
| 美妆护肤 | [萌萌哒R] [笑哭R] [拿走R] [比心R] |
| 旅行攻略 | [打卡R] [派对R] [冲鸭R] [炸裂R] |
| 美食探店 | [吃瓜R] [笑哭R] [赞R] [拿走R] |
| 好物分享 | [火R] [赞R] [拿走R] [比心R] [冲鸭R] |

## ⚠️ 常见问题

### Q1: 为什么我看到的是文本代码而不是图片？

**A**: 这是正常的！
- **网页中**：显示为 `[笑哭R]` 文本
- **小红书 app 中**：自动显示为 😂 图片

### Q2: AI 添加的表情太多/太少？

**A**: 使用手动添加功能几次，系统会学习你的偏好：
1. 点击"添加小红书表情"
2. 调整到你喜欢的数量和位置
3. 保存
4. 之后 AI 会参考你的风格

### Q3: 如何关闭 AI 自动添加表情？

**A**: 暂时不支持关闭，但你可以：
- 在手动编辑时删除不想要的表情
- 系统会记住你的偏好，逐渐减少使用

### Q4: 表情选择器有哪些分类？

**A**: 6 大分类：
- 🔥 热门推荐（最常用的 12 个）
- 😊 表情类
- 👋 手势类
- 🎯 动作类
- 🎁 物品类
- ✨ 其他

### Q5: 可以搜索表情吗？

**A**: 可以！在表情选择器顶部输入关键词：
- 搜"笑" → 找到笑相关的表情
- 搜"赞" → 找到点赞相关的表情
- 搜"可爱" → 找到萌萌哒等表情
