# 快捷指令 - 文本拼接方式（最简单）

## 如果找不到「词典」选项，用这个方法

### 完整步骤

#### 操作1-5：前面保持不变
```
1. 获取剪贴板
2. 从菜单中选择（法意瑞/其他国家）
3-5. If/Else 设置 appToken 和 tableId 变量
6. 结束菜单
```

#### 操作6：创建 JSON 文本（替代词典）

**删除「设定变量 requestBody」，改用「文本」操作：**

1. 在「结束菜单」后面，点击 `+`
2. 搜索并添加「**文本**」操作
3. 在文本编辑器中，输入以下内容：

```json
{
  "url": "这里插入剪贴板变量",
  "appToken": "这里插入appToken变量",
  "tableId": "这里插入tableId变量",
  "async": true
}
```

**重要**：上面的文字只是示意，实际操作如下：

##### 步骤A：输入 JSON 框架
先输入这个框架（纯文本）：
```
{
  "url": "",
  "appToken": "",
  "tableId": "",
  "async": true
}
```

##### 步骤B：插入变量
然后在对应位置插入变量：

1. **url 的值**：
   - 将光标放在 `"url": ""` 的两个引号**之间**
   - 点击键盘上方的「变量」按钮（或长按输入框）
   - 选择「**剪贴板**」变量

2. **appToken 的值**：
   - 将光标放在 `"appToken": ""` 的两个引号之间
   - 插入「**appToken**」变量

3. **tableId 的值**：
   - 将光标放在 `"tableId": ""` 的两个引号之间
   - 插入「**tableId**」变量

4. **async** 保持 `true` 不变（纯文本）

##### 完成后的样子：
```
{
  "url": "[剪贴板]",
  "appToken": "[appToken]",
  "tableId": "[tableId]",
  "async": true
}
```
注意：`[变量]` 会显示为蓝色/橙色的标签

#### 操作7：获取 URL 内容
1. 添加「获取 URL 内容」操作
2. **URL**：`https://content-factory-jade-nine.vercel.app/api/xiaohongshu/quick-save`
3. **方法**：`POST`
4. **请求体**：选择「**JSON**」
5. **JSON 内容**：选择「**文本**」变量（就是上一步创建的 JSON 文本）

---

## 🎯 关键点

1. 先输入 JSON 文本框架
2. 再在引号之间插入变量
3. 变量会显示为彩色标签，不是普通文字
4. 最后将这个文本作为 JSON 发送

---

## 完整流程（文本拼接方式）

```
1. 获取剪贴板
2. 从菜单中选择
3-5. If/Else 设置变量
6. 结束菜单
7. 文本（JSON 格式，包含变量）  ← 新方法
8. 获取 URL 内容（JSON = 文本）
9. 从输入获取词典值 message
10. 显示通知
```

---

这个方法不需要找「词典」选项，直接用文本拼接 JSON，更简单！
