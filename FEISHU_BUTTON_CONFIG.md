# 飞书"去复刻"按钮配置指南

## 问题现象
点击"去复刻"按钮没有跳转到内容工厂的复刻页面。

## 配置步骤

### 1. 检查按钮字段名称
确保飞书表格中的按钮字段名称为：`去复刻`（必须完全匹配）

### 2. 按钮字段配置

在飞书表格中，按钮字段应该配置为：

**字段类型**：按钮

**按钮设置**：
- **按钮类型**：打开链接
- **按钮文字**：去复刻
- **链接地址**：
  ```
  https://xiaohongshu-collect.vercel.app/rewrite?title={{标题}}&content={{正文}}&tags={{话题标签}}
  ```
- **打开方式**：✅ 在新标签页打开

### 3. 字段引用格式

⚠️ **重要**：链接中的字段引用必须使用双大括号 `{{字段名}}`，并且字段名必须与表格中的列名**完全匹配**。

正确示例：
- `{{标题}}` → 引用"标题"列
- `{{正文}}` → 引用"正文"列
- `{{话题标签}}` → 引用"话题标签"列

### 4. 常见问题排查

#### 问题1：按钮没有反应
**原因**：字段名称不匹配
**解决**：检查链接中的 `{{标题}}`、`{{正文}}`、`{{话题标签}}` 是否与表格列名完全一致

#### 问题2：跳转后没有数据
**原因**：字段引用错误或字段为空
**解决**：
1. 检查表格中是否有"标题"、"正文"、"话题标签"三列
2. 检查这些列中是否有数据
3. 检查列名是否完全匹配（包括空格、中英文）

#### 问题3：URL中有乱码
**原因**：飞书会自动对URL进行编码
**解决**：这是正常的，不需要处理

### 5. 测试方法

1. 在表格中找到一条有完整数据的记录（标题、正文、话题标签都有内容）
2. 点击该记录的"去复刻"按钮
3. 应该会在新标签页打开复刻页面，并且标题、正文、话题标签都已预填充

### 6. 完整配置示例

假设您的表格列顺序是：
1. 笔记链接
2. 作者昵称
3. 封面
4. 图片2
5. 后续图片
6. **标题**
7. **正文**
8. **话题标签**
9. 浏览数
10. 点赞数
11. 收藏数
12. 评论数
13. 发布时间
14. **去复刻**（按钮字段）

按钮配置的链接地址：
```
https://xiaohongshu-collect.vercel.app/rewrite?title={{标题}}&content={{正文}}&tags={{话题标签}}
```
