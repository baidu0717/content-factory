# 小红书笔记发布检查清单

> 发布前，用这个清单检查一遍，避免常见错误。

---

## 📝 内容检查

### 标题
- [ ] 长度适中（15-25 字）
- [ ] 身份标签明确（大学生、宝妈等）
- [ ] 关键词前置
- [ ] 数字具体（不用模糊词）
- [ ] 有吸引力（痛点/结果/好奇）
- [ ] 与内容匹配（不标题党）

### 开头（前 50 字）
- [ ] 快速抓人（3 秒内吸引注意）
- [ ] 痛点共鸣或结果展示
- [ ] 不拖沓，直入主题

### 正文
- [ ] 结构清晰（有小标题或序号）
- [ ] 段落简短（每段 2-3 行）
- [ ] 有具体案例或数据
- [ ] 有行动指导（可执行）
- [ ] 视觉优化（emoji、分隔符）
- [ ] 避免空洞说教
- [ ] 字数适中（300-600 字）

### 结尾
- [ ] 有总结或要点回顾
- [ ] 有行动引导（收藏/评论/关注）
- [ ] 语气友好，不生硬

---

## 🏷️ 标签检查

### 标签数量
- [ ] 5-8 个标签
- [ ] 不少于 5 个
- [ ] 不超过 10 个

### 标签类型
- [ ] 有领域标签（1-2 个）
- [ ] 有人群标签（1 个）
- [ ] 有场景或内容标签（1-2 个）

### 标签质量
- [ ] 与内容高度相关
- [ ] 不蹭不相关的热门标签
- [ ] 没有重复意思的标签
- [ ] 热度搭配合理（高中低都有）

### 推荐组合
- [ ] 参考了标签库的成功组合
- [ ] 标签顺序：核心标签在前

---

## 🎨 封面检查

### 图片
- [ ] 清晰度高
- [ ] 与内容相关
- [ ] 不喧宾夺主

### 文字
- [ ] 标题可见（一眼能看清）
- [ ] 字号够大
- [ ] 颜色对比度强
- [ ] 关键词突出（加粗/变色/放大）

### 整体
- [ ] 视觉美观
- [ ] 排版整齐
- [ ] 不信息过载
- [ ] 符合小红书风格

---

## 🎯 对象匹配检查

### 目标对象
- [ ] 明确谁是目标对象
- [ ] 标题中有身份标签
- [ ] 内容切中对象痛点
- [ ] 语言风格符合对象偏好

### 价值提供
- [ ] 提供了明确价值
- [ ] 解决了实际问题
- [ ] 有可执行的建议
- [ ] 不空洞不说教

---

## ✅ 质量自查

### 基本要求
- [ ] 没有错别字
- [ ] 没有语句不通
- [ ] 没有低级错误（数字、时间等）
- [ ] 没有敏感词汇

### 内容质量
- [ ] 信息准确（数据、案例真实）
- [ ] 逻辑清晰（前后连贯）
- [ ] 有独特价值（不是抄袭或烂大街的内容）
- [ ] 真实不做作（不过度包装）

### 用户体验
- [ ] 易读（排版、字号、颜色）
- [ ] 易懂（表达清晰，不拗口）
- [ ] 有用（能解决问题或提供价值）

---

## 🚀 发布准备

### 发布时间
- [ ] 选择了合适的发布时间
  - 推荐：7-9 点（上班/上学路上）

  - 推荐：18-22 点（下班后）

### 发布后

- [ ] 设置消息提醒
- [ ] 准备好互动内容

## 💡 常见错误避免

### 标题错误
- ❌ 信息过载（堆砌太多信息）
- ❌ 标题党（内容做不到）
- ❌ 模糊表达（"很多""非常""超级"）
- ❌ 关键词后置（重要信息在后面）

### 内容错误
- ❌ 开头拖沓（50 字还没进入主题）
- ❌ 结构混乱（没有小标题和序号）
- ❌ 空洞说教（没有具体案例）
- ❌ 排版密集（大段文字没有分段）

### 标签错误
- ❌ 只用热门标签（不精准）
- ❌ 蹭不相关流量（被限流）
- ❌ 标签重复（浪费名额）
- ❌ 标签太少或太多（影响曝光）

## 📋 发布流程

### Step 1：内容最终检查
- 用本清单过一遍
- 修改不合格的地方

### Step 2：封面最终检查
- 3 秒测试（自己看 3 秒，能看懂吗？）
- 手机预览（小屏幕上看得清吗？）

### Step 3：标签最终检查
- 参考标签库
- 确认相关性
- 确认数量

### Step 4：发布
- 选择合适的时间
- 点击发布

### Step 5：发布后
- 回复第一批评论（提高互动率）
- 记录发布时间
- 设置提醒（24h 后查看数据）

---

## 🎯 检查清单使用技巧

### 第一次使用
- 完整过一遍所有项目
- 标记不合格的地方
- 逐项修改

### 熟练后
- 重点检查常犯错误
- 快速过一遍核心项
- 5 分钟内完成检查

### 定期回顾
- 每周回顾一次清单
- 根据实际情况调整
- 删除不重要的项
- 添加常犯错误项

---
