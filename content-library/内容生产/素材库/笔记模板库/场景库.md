## 瑞士景点

- 戈纳格拉特（Gornerglatt，3089米）：全程坐登山齿轨火车（Gornergrat Bahn），无需步行，老人可承受，山顶直面马特洪峰全景，是瑞士最经典的打卡机位之一
- First 山：海拔低于少女峰，可步行徒步，老人有高反时可替代少女峰，站在 First 山对面就是少女峰，景色不打折扣

## 小团专属场景

> 以下场景仅在小团/定制小团环境下发生，定制自由行笔记里没有。写E型小团版时优先调用。

- 小团vs大团：人少时间安排更自由，吃饭可以一起商量，不吃强制团餐；全程MPV接送，比大团大巴舒适（来源：真实客人反馈/20251027-6人小团）
- 小团vs自由行：MPV接送比公共交通舒适便利，大城市停车难无需考虑；出国人生地不熟，小团安全保障更强（来源：真实客人反馈/20251027-6人小团）
- 小团可以自己选机票：团友各自订，可以选便宜公务舱，大团没有这个自由（来源：真实客人反馈/20251027-6人小团）
- 同团都是相识的人，不用将就陌生人节奏；经过一段旅程同团成了朋友（来源：真实客人反馈/20251027-6人小团）
- 小团司导带去百年玻璃作坊看穆拉诺玻璃手工制作——大团不安排的小众体验（来源：定制和小团/20260308-带妈妈纯玩小团）
- 华人司导10年以上欧洲经验，全程帮所有团员拍照，人少所以每个人都有大片（来源：定制和小团/20260308-带妈妈纯玩小团）
- 威尼斯遇到意大利大罢工，司导现场协调安排成功上岛（来源：真实客人反馈/20251204-带3岁娃）
- 司导知道当地哪里好吃：带去本地人爱去的餐厅，避开旅游区宰客价；当推荐的餐厅当晚满座，几分钟内推了另一家同样好吃的（来源：定制和小团/20260308；定制自由行/20260304）
- 小团行程临时商量加景点：朋友说某个地方很好玩，司导知道后主动加入行程，不用客人开口要（来源：定制和小团/20240609-一家子瑞士YYDS）

## 带老人出行

- 老人腰不太好，担心长途坐车不舒服 → 全程安排商务车，行程不赶、每段坐车时间不长，全程跟下来没有不适（来源：定制和小团/20241016-带68岁父母法意瑞）
- 有8年腰椎病 → 真皮座商务车，全程坐得很舒适，一点不感觉累；定制时告知身体情况，司导途中主动安排休息站（来源：定制和小团/20240914-带妈妈法意瑞4W）
- 罗马酒店安排在斗兽场步行800米内 → 上午逛完景点直接溜达回酒店睡午觉，下午精力恢复再去万神殿；老人不需要全天赶路（来源：定制和小团/20250222-第一次带父母出国）
- CityWalk 自由活动前，司导详细交代防盗注意事项；自由活动期间随时待命，偶尔发消息关心，但不催促（来源：定制和小团/20250222-第一次带父母出国）
- 老人登少女峰：定制师出发前主动询问有无高反/恐高。无明显高反的：司导开车送到山脚缆车站，坐火车+缆车上去，司导全程陪同，注意提前戴墨镜，高反程度一般较轻。有明显高反的：改去 First 山，海拔更低、可简单徒步，站在 First 山对面就是少女峰，景色不打折扣。
- 带老人出行之前拒绝长途旅游 → 回来后主动问下次去哪玩（前后转变，说明定制游体验超出预期）
- 父亲有轻微高血压，妈妈怕长时间走路 → 定制师增加一天休整、标注每个景点步行强度、每隔两三天安排半天轻松行程（来源：定制和小团/20250415-4w+法意瑞已回）
- 定制师根据妈妈「爱拍照、不能太累、喜欢文化景点」三个具体需求规划路线，所有预约（门票、火车、餐厅）全部搞定（来源：定制自由行/20260225-法意瑞十天）
- 每天行程内置午休缓冲：妈妈中午可以回酒店休息一小时，下午精力恢复再出发，全程跟下来没有不适（来源：定制自由行/20260225-法意瑞十天）
- 酒店按位置选：巴黎酒店出门就是老牌甜品店，罗马离许愿池步行15分钟，因特拉肯开窗就是雪山湖景（来源：定制自由行/20260225-法意瑞十天）
- 穿插施皮茨小镇、龙疆湖等安静出片的小众地点，不只是热门景点打卡，节奏轻松且出片（来源：定制自由行/20260225-法意瑞十天）
- 巴黎酒店指定有埃菲尔铁塔景的房型，父母激动得睡不着觉 [待补充：具体酒店名称]（来源：定制和小团/20250415-4w+法意瑞已回）
- 瑞士木屋民宿，推窗就是雪山，早餐有自制果酱，房东热情像亲戚（来源：定制和小团/20250415-4w+法意瑞已回）

## 定制自由行专项服务

- 行程同步到专属 App + 行程手册双保险：酒店地址、交通方式、地铁出口编号、站台编号、景点间步行路线全部标注，客人跟着走即可（来源：定制自由行/20251112-意大利10天；定制自由行/20260203-蜜月后续）
- 城市间高铁：顾问讲清楚票怎么买划算、站台怎么找、转车留多少缓冲时间（来源：定制自由行/20251112-意大利10天）
- 欧洲城市间火车票提前买好，并指导如何认站台、转车缓冲时间留多少，不需要客人自己研究（来源：定制自由行/20260225-法意瑞十天）
- 突发情况：当地行程达人/24H在线顾问随时待命，遇到变故快速给出 Plan B；实际案例：客人戒指落在飞机上，顾问根据行程逐一排查找回（来源：定制自由行/20251112-意大利10天；定制自由行/20260203-蜜月后续）
- 知道客人爱拍照 → 额外安排菲罗雷峡谷桥、罗马中心花园等人少有特色小众地点（来源：定制自由行/20251112-意大利10天）
- 酒店按位置选：佛罗伦萨挨乌菲兹美术馆、罗马对望西班牙广场、托斯卡纳推窗田园风光（来源：定制自由行/20251112-意大利10天）

## 难票蹲守

- 《最后的晚餐》门票极难，提前半年可能也没有 → 定制师盯着官网退票时段，手快蹲到零散名额，没有内部渠道，就是熬（来源：3.18笔记）

## 饮食适配 / 民宿安排

- 担心老人吃不惯西餐 → 定制师在瑞士安排民宿，可自己买菜做饭；采尔马特民宿早上热泡面，抬头窗外就是雪山（来源：3.18笔记）

## VIP通道 / 避开排队

- 凡尔赛宫、卢浮宫旺季排队可达1-2小时 → 提前购买VIP门票或预约特定入场时段，走专属通道直接进入，节省大量时间（来源：定制和小团/20250527-6人小团）
- 卢浮宫：早上一开馆就入场 → 蒙娜丽莎展厅人少，看完出来时外面已排起长龙；老人不需要长时间排队站立（来源：定制和小团/20260308-带妈妈纯玩小团）

## 带孩子出行

- 一家四口法意瑞12天，人均2.6w含酒店/机票/门票/签证/交通/保险，自理正餐，无额外消费（费用结构清晰，精算型家庭的参考锚点）（来源：定制自由行/20260225-一家四口法意瑞）
- 家庭成员兴趣不同：妈妈/爸爸/女儿/儿子分别有各自满意点 → 定制行程按每人喜好安排，女儿爱美术馆多排几个，儿子爱Gelato安排充足自由觅食时间（来源：定制自由行/20260225-一家四口法意瑞）
- 签正规合同，全程无强制购物，一个购物店都没进，买礼品全凭自己高兴（对带家人出行者消除购物坑顾虑）（来源：定制自由行/20260225-一家四口法意瑞）
- 不用拼团陌生人，全家独享包车，节奏自定，爸爸有心思和精力给孩子拍各种开心游客照（来源：定制自由行/20260225-一家四口法意瑞）
- 一家四口意瑞10天，人均2.7w全包（定制费+机票+酒店+签证+门票+交通），只管吃饭和购物，50+家长第一次带全家出国（来源：定制自由行/20260227-一家四口意瑞）
- 家庭成员需求完全不同（妈妈要雪山徒步/老公要米兰大教堂/女儿要苏黎世美术馆/儿子要巧克力工厂）→ 定制师一个需求没落下，包车+瑞士景观火车组合（来源：定制自由行/20260227-一家四口意瑞）
- 定制师帮预约苏黎世美术馆特展（难抢票的特展提前帮约好，客人只需告知兴趣方向）（来源：定制自由行/20260227-一家四口意瑞）
- 不用吃团餐，定制师推荐当地餐厅：瑞士奶酪火锅、意大利地道T骨牛排，每天换着花样吃（儿子最满意）（来源：定制自由行/20260227-一家四口意瑞）
- 带11岁历史迷男孩：罗马多留一天，安排专为孩子设计的斗兽场互动讲解（孩子听得进去，非成人导览式讲解）（来源：定制自由行/20260227-带娃意大利）
- 梵蒂冈博物馆暑假排队2小时起步 → 提前预约快速通道票，直接入场，保护孩子体力（来源：定制自由行/20260227-带娃意大利）
- 托斯卡纳自驾迷路 → 定制师在线实时指导到找到路，客人不需要自己解决陌生地方的突发状况（来源：定制自由行/20260227-带娃意大利）
- 10天意大利路线：罗马3晚→佛罗伦萨2晚（一天自驾托斯卡纳）→威尼斯2晚→米兰2晚；每城住够2晚，节奏慢，儿子累了随时回酒店休息（来源：定制自由行/20260227-带娃意大利）
- 挑定制师三步筛选法：①淘汰不专业的（不问孩子喜好、合同随意）②看谁真懂带孩子（行程不能太满、梵蒂冈要有快速通道）③核实真实服务（要之前带孩子家庭的反馈、看酒店实拍和App截图）（来源：定制自由行/20260227-带娃意大利）
- 攻略无能经典路径：刷了两周攻略越刷越焦虑（景点预约/城市交通/找路），英语早还给老师，头铁自由行放弃 → 找定制解决（来源：定制自由行/20260304-意大利亲子游）
- 推荐的当地牛排馆当晚满座 → 微信群联系，几分钟推了另一家本地人爱去的，味道同样好（行程中餐饮应急推荐，无需客人自己临时搜索）（来源：定制自由行/20260304-意大利亲子游）

## 蜜月情侣

### 蜜月定制自由行专项

- 担心两人公共交通迷路 → 行程手册细致到地铁坐哪站、怎么走、怎么买票，全程照着走（来源：定制自由行/20260325-纯自来水安利）
- 睡眠浅提前告知顾问 → 安排隔音好的酒店房间（个人身体需求可以提前定制）（来源：定制自由行/20260325-纯自来水安利）
- 定制不含餐，懒得找早餐 → 让顾问安排含早酒店；不知道吃啥随时发消息让顾问推荐，比自己找网红店好（来源：定制自由行/20260325-纯自来水安利）
- 十日意大利蜜月路线：佛罗伦萨→托斯卡纳→罗马→阿玛菲，节奏松弛；包含酒店/机票/景点门票/签证/火车票/保险（来源：定制自由行/20260325-纯自来水安利）
- 佛罗伦萨酒店推窗就能拍到圣母百花大教堂（酒店选景观位置，蜜月氛围感自然有）（来源：定制自由行/20260325-纯自来水安利）
