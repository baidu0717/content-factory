# 小红书内容库 - AI 使用指南

## 项目简介

这是一个系统化的小红书内容生产工作流，帮助你：
- 高效管理选题和素材
- 复用已验证的爆款内容
- 基于对象分析精准创作
- 数据驱动持续迭代

## 核心理念：从碎片到系统

**传统碎片化创作**：
- 有想法 → 问 AI → 写文案 → 发布 → 忘记
- 每次都从零开始，重复造轮子

**系统化创作**：
- 有想法 → 记录选题 → 检索素材库 → 分析对象 → 复用框架 → 发布 → 数据复盘 → 沉淀方法论
- 每次创作都在给系统添砖加瓦

## 完整工作流

### 1. 选题记录（碎片想法管理）

当你有一个想法时，对我说：**「记录选题」**

我会自动把想法记录到 `01-内容生产/选题管理/00-选题记录.md`

### 2. 对象分析（精准定位）

当你要为某个对象创作内容时，告诉我：**「分析对象：[对象描述]」**

我会：
1. 检索 `02-对象库/` 查看是否已有该对象的分析
2. 检索 `01-内容生产/素材库/对象分析模板.md` 找到合适的分析框架
3. 生成对象分析报告并保存到 `02-对象库/对象档案/`

### 3. 素材检索（避免重复造轮子）

我会自动检索素材库：
1. **爆款笔记库**：有没有类似的已发布笔记？
2. **标题库**：有没有适合的标题模板？
3. **金句库**：有没有高质量的表达？
4. **对象分析模板**：有没有相关的对象分析框架？

如果找到相关内容，我会建议你复用，而不是重新创作。

### 4. 内容创作（基于素材和对象）

根据对象分析和素材库，我会帮你：
1. 生成文案结构
2. 推荐标题选项（基于 `方法论沉淀/小红书标题方法论.md`）
3. 建议话题标签（基于 `素材库/话题标签库.md`）
4. 提供封面建议（基于 `方法论沉淀/小红书封面方法论.md`）

内容会保存到 `01-内容生产/选题管理/02-创作中的选题/`

### 5. 素材保存（建立内容资产）

当你要保存一篇完整的小红书笔记时，对我说：**「记录素材」**

然后提供：
- 笔记标题
- 笔记正文
- 话题标签
- 图片（封面 + 内页图片）

我会：
1. 在 `素材库/爆款笔记库/` 创建以标题命名的文件夹
2. 保存所有图片（封面.jpg、内页图片/）
3. 保存文案内容（文案.md）
4. 提取金句到 `金句库.md`
5. 提取标题模式到 `标题库.md`
6. 提取话题标签到 `话题标签库.md`

### 6. 发布与数据记录

发布后，告诉我：**「数据复盘」**

我会：
1. 把笔记移动到 `03-已发布的笔记/`
2. 记录数据表现（点赞、收藏、评论、分享）
3. 更新 `内容数据统计/笔记数据统计表.xlsx`
4. 分析爆款规律，更新 `内容数据统计/爆款笔记分析.md`
5. 反哺到 `方法论沉淀/`

## AI 自动化能力

我具备以下自动化能力（定义在 `skills/` 目录）：

1. **记录选题**：快速记录碎片想法
2. **记录素材**：保存完整的小红书笔记（图片+文案+标签）
3. **检索素材**：自动查找可复用的内容
4. **分析对象**：分析目标对象，生成内容策略
5. **生成标题**：基于方法论生成小红书标题
6. **生成文案**：基于素材库和对象分析生成文案
7. **推荐话题**：根据内容推荐合适的话题标签
8. **数据复盘**：记录发布数据并分析规律

## 使用原则

### 1. 先检索，再创作

每次创作前，我都会先检索素材库：
- 避免重复内容
- 复用已验证的框架
- 保持内容风格一致

### 2. 边用边沉淀

- 好的标题 → 沉淀到标题库
- 好的表达 → 沉淀到金句库
- 好的笔记 → 完整保存到爆款笔记库
- 好的数据 → 反哺到方法论

### 3. 数据驱动迭代

- 记录每篇笔记的数据表现
- 分析爆款笔记的共同规律
- 持续更新方法论

### 4. 对象导向创作

- 先分析对象特征
- 再检索相关素材
- 最后精准创作内容

## 文件命名规范

### 笔记文件夹命名
- 格式：`YYYYMMDD-笔记标题`
- 示例：`20260208-如何用小红书月入过万`

### 图片命名
- 封面：`封面.jpg`
- 内页：`内页1.jpg`、`内页2.jpg`...

### 对象档案命名
- 格式：`[对象类型]-[核心特征].md`
- 示例：`大学生-转行互联网.md`、`宝妈-副业赚钱.md`

## 注意事项

1. **素材库是核心**：每次创作都要先检索素材库
2. **对象分析是前提**：了解对象才能精准创作
3. **数据驱动迭代**：持续记录数据，反哺方法论
4. **内容是资产**：好的内容要完整保存，方便复用

## 系统价值

这套系统的核心价值不是「AI 能帮我写文案」，而是：

1. **记忆系统**：AI 知道你以前写过什么，避免重复造轮子
2. **素材复用**：好的框架、表达、案例可以反复使用
3. **方法论沉淀**：每次创作都在给系统添砖加瓦
4. **数据驱动**：不凭感觉，凭数据迭代
5. **对象精准**：基于对象分析，精准创作内容
