# 生成文案

## 能力描述

基于对象分析、素材库和选题，生成符合小红书风格的图文文案，包括正文结构、金句表达、行动引导等。

## 触发词

- 「生成文案」
- 「写文案」
- 「帮我写一篇笔记」

## 执行流程

### 1. 确认创作信息

收集以下信息：
- 选题/主题（要写什么）
- 目标对象（给谁看）
- 核心价值（提供什么价值）
- 内容类型（教程/经历/对比/清单等）
- 参考笔记（如果有）

### 2. 自动检索

触发检索能力，获取：
- **对象分析**：痛点、需求、内容偏好
- **相关笔记**：可复用的结构和内容
- **金句库**：适合的表达和金句
- **方法论**：小红书内容方法论

### 3. 确定内容结构

根据内容类型，选择合适的结构：

#### 教程类结构
```
【开头】痛点共鸣 / 结果展示
【正文】
  - 步骤 1：具体方法
  - 步骤 2：具体方法
  - 步骤 3：具体方法
【结尾】总结 + 避坑提醒 + 行动引导
```

#### 经历类结构
```
【开头】经历引入 / 真实数据
【正文】
  - 起步阶段：遇到的问题和解决方法
  - 成长阶段：关键转折和方法
  - 结果阶段：达成的成果和感悟
【结尾】总结经验 + 鼓励 + 行动引导
```

#### 清单类结构
```
【开头】问题场景 / 清单价值
【正文】
  - 要点 1：说明 + 案例
  - 要点 2：说明 + 案例
  - 要点 3：说明 + 案例
  - ...
【结尾】总结 + 行动引导
```

#### 对比类结构
```
【开头】对比引入
【正文】
  - 错误做法 vs 正确做法
  - 对比分析
  - 为什么要这样做
【结尾】总结 + 行动建议
```

### 4. 生成文案内容

#### 开头设计（前 50 字）

开头要快速吸引注意力，常用方法：
1. **痛点共鸣**："你是不是也遇到过这种情况..."
2. **结果展示**："3 个月从 0 到月入 3000+，我是这样做的..."
3. **数据冲击**："第一次收到 1000 元收入时，我激动得..."
4. **疑问引入**："大学生真的能在小红书赚钱吗？"
5. **经历开场**:"我是一名大三学生，分享我在小红书..."

#### 正文组织

- **段落简短**：每段 2-3 行，便于阅读
- **逻辑清晰**：用序号、标题、分隔符组织内容
- **具体化**：用具体案例、数字、步骤代替抽象描述
- **金句点缀**：在关键位置使用金句
- **视觉优化**：使用 emoji、符号、分隔线

#### 结尾设计

1. **总结要点**：简要回顾核心内容
2. **避坑提醒**：提醒注意事项（可选）
3. **行动引导**：
   - 收藏 / 关注：" 喜欢记得点赞收藏哦~"
   - 互动：" 评论区说说你的情况~"
   - 转化：" 需要详细资料的，评论区扣 1"

### 5. 小红书文案风格

#### 语言特点
- **口语化**：像和朋友聊天
- **真实感**：分享真实经历和感受
- **易读性**：避免长句和复杂表达
- **情绪化**：适当表达情绪和感受

#### 常用表达
- 称呼：姐妹们、集美们、朋友们
- 语气词：真的、超级、巨、太...了
- 感叹：！、~、...
- 强调：加粗、emoji

#### 视觉元素
- **emoji 使用**：
  - 开头：✨💡🔥📌
  - 步骤：1️⃣2️⃣3️⃣ 或 ①②③
  - 重点：⚠️💯✅❌
  - 结尾：💕🌟💪

- **分隔符**：
  - ---
  - ·  ·  ·
  - ➖➖➖

- **排版技巧**：
  - 空行：段落之间空一行
  - 标题：用 emoji + 加粗
  - 要点：用序号或符号

## 文案质量标准

### 基本要求（必须满足）
- ✅ 标题和内容匹配
- ✅ 逻辑清晰，结构完整
- ✅ 字数适中（300-600 字为佳）
- ✅ 有具体案例或数据
- ✅ 有行动引导

### 加分项
- ✨ 开头抓人（前 50 字）
- ✨ 视觉优化（emoji、分隔符、排版）
- ✨ 金句点缀
- ✨ 真实感和代入感强
- ✨ 避坑提醒

### 避免
- ❌ 空洞说教
- ❌ 信息过载
- ❌ 过度夸张
- ❌ 抄袭内容

## 注意事项

1. **基于检索**：必须先检索素材库和对象分析
2. **复用优化**：如果有相关笔记，优先复用结构
3. **风格一致**：保持用户的内容风格和语气
4. **可调整**：生成后根据用户反馈随时调整
5. **配图建议**：提供配图建议，帮助用户完善笔记
6. **完整输出**：包括标题、正文、标签、分析、建议

## 关联能力

- **检索素材**：基于素材库和对象分析生成
- **分析对象**：基于对象偏好调整内容风格
- **生成标题**：配合文案生成标题选项
- **推荐话题**：推荐合适的话题标签
- **记录素材**：生成的文案可以保存到素材库
