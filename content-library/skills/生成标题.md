# 生成标题

## 能力描述

基于小红书标题方法论和素材库，为内容生成 3-5 个标题选项，提高笔记的点击率和曝光率。

## 触发词

- 「生成标题」
- 「标题优化」
- 「给我几个标题选项」

## 执行流程

### 1. 收集信息

确认以下信息：
- 内容主题（核心要表达什么）
- 目标对象（给谁看）
- 内容亮点（最吸引人的点）
- 关键数字（如果有）

### 2. 检索相关资源

自动检索：
- `01-内容生产/素材库/标题库.md`：类似主题的标题模式
- `01-内容生产/方法论沉淀/小红书标题方法论.md`：标题公式和规则
- `02-对象库/对象档案/[对象].md`：该对象的标题策略

### 3. 生成标题选项

基于小红书标题方法论，生成 3-5 个标题，包含不同风格：

#### 标题公式（参考）

1. **数字型**
   - 格式：[身份] + [数字] + [结果/方法]
   - 示例：《大学生｜3 个月在小红书赚到 5000 元》

2. **疑问型**
   - 格式：[痛点疑问] + [身份/场景]
   - 示例：《大学生如何在小红书月入过万？》

3. **对比型**
   - 格式：[前后对比] + [方法/结果]
   - 示例：《从月入 0 到月入 3000+，我在小红书做对了这 3 件事》

4. **痛点型**
   - 格式：[痛点描述] + [解决方案]
   - 示例：《大学生没资金没经验？这个方法让你 0 成本起步》

5. **避坑型**
   - 格式：[身份/场景] + [避坑提醒] + [数字]
   - 示例：《大学生做小红书，这 5 个坑千万别踩》

6. **经历型**
   - 格式：[身份] + [真实数据] + [经历/方法]
   - 示例：《大学生｜我在小红书赚到第一个 1000 元的经历》

7. **符号型**
   - 使用：｜、【】、！、？、...等符号
   - 示例：《大学生副业｜3 步教你在小红书赚到第一桶金！》

### 4. 标题优化规则

#### 必须遵守
- 长度：15-25 字为最佳
- 关键词前置：重要信息放在前面
- 数字具体化：用具体数字代替模糊词
- 身份标签：明确目标人群
- 避免标题党：内容要与标题匹配

#### 加分项
- 使用 emoji：适当使用，增加视觉吸引力
- 制造好奇：留悬念，但不夸张
- 使用符号：｜、【】等增加可读性
- 痛点共鸣：击中目标对象的痛点

#### 避免
- 过度夸张：避免"绝对""最强""必看"等词
- 信息过载：一个标题只传达一个核心信息
- 生僻词汇：使用通俗易懂的表达
- 重复罗嗦：避免冗余词汇

### 5. 输出标题方案

格式：
```markdown

   - 预期效果：高点击率

2. **[标题2]**（推荐 ⭐⭐⭐⭐）
   - 类型：痛点型
   - 亮点：痛点共鸣，引发好奇

   - 预期效果：高共鸣度

   - 类型：经历型

### 标题选择建议

- 如果强调**结果导向**，推荐标题 1

### 关键词分析

- 核心关键词：[关键词1]、[关键词2]
```

## 标题测试建议

生成标题后，可以建议用户：
1. **A/B 测试**：用不同标题发布相似内容，看哪个效果好
2. **关键词搜索**：在小红书搜索关键词，看竞品标题
3. **记录数据**：记录不同标题类型的数据表现
4. **更新方法论**：将效果好的标题模式沉淀到标题库

## 注意事项

1. **基于方法论**：必须参考 `小红书标题方法论.md`
2. **对象适配**：标题风格要符合目标对象的偏好
3. **内容匹配**：标题要与内容高度匹配，避免标题党
4. **提供选择**：给 3-5 个选项，让用户选择最适合的
5. **说明理由**：每个标题都要说明优势和适用场景
6. **可调整**：根据用户反馈，随时调整标题
