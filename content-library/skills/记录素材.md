# 记录素材

## 能力描述

当用户说「记录素材」或提供一篇完整的小红书笔记时，自动保存笔记的所有内容（图片+文案+标签），并提取可复用的元素到素材库。

## 触发词

- 「记录素材」
- 「保存笔记」
- 「添加到素材库」

## 执行流程

### 1. 收集信息

询问用户提供以下信息：
- 笔记标题
- 笔记正文
- 话题标签（多个）
- 封面图片
- 内页图片（如有）
- 数据表现（点赞、收藏、评论、分享）- 可选

### 2. 创建笔记文件夹

在 `01-内容生产/素材库/爆款笔记库/` 创建文件夹：
- 命名格式：`YYYYMMDD-笔记标题`
- 示例：`20260208-如何用小红书月入过万`

### 3. 保存内容

在笔记文件夹内创建：
- `文案.md`：包含标题、正文、话题标签、数据表现
- `封面.jpg`：封面图片（如果用户提供）
- `内页图片/`：内页图片文件夹（如有）

文案.md 格式：
```markdown
# [笔记标题]

## 标题
[标题内容]

## 正文
[正文内容]

## 话题标签
- #标签1
- #标签2
- #标签3

## 数据表现（如有）
- 发布时间：YYYY-MM-DD
- 点赞：XXX
- 收藏：XXX
- 评论：XXX
- 分享：XXX

## 笔记亮点分析
[分析这篇笔记为什么表现好，提取可复用的元素]
```

### 4. 提取可复用元素

#### 提取标题模式

分析标题的结构和特点，添加到 `01-内容生产/素材库/标题库.md`：
- 标题类型（如：数字型、疑问型、对比型）
- 标题公式（如：[身份] + [痛点] + [结果]）
- 示例标题

#### 提取金句

找出正文中的金句（有价值、可复用的表达），添加到 `01-内容生产/素材库/金句库.md`：
- 分类（如：痛点描述、解决方案、行动号召）
- 金句内容
- 适用场景

#### 提取话题标签

添加到 `01-内容生产/素材库/话题标签库.md`：
- 标签分类（如：领域标签、人群标签、场景标签）
- 标签名称
- 标签热度（如有数据）
- 适用主题

### 5. 确认完成

告诉用户：
- 笔记已保存到哪个文件夹
- 提取了哪些可复用元素
- 素材库已更新

## 示例对话

**用户**：记录素材

**AI**：好的，请提供以下信息：
1. 笔记标题
2. 笔记正文
3. 话题标签
4. 图片（可选）
5. 数据表现（可选）

**用户**：[提供内容]

**AI**：
已保存笔记到：`01-内容生产/素材库/爆款笔记库/20260208-如何用小红书月入过万/`

提取的可复用元素：
- 标题模式：数字型 + 结果导向
- 金句 3 条：已添加到金句库
- 话题标签 5 个：已添加到话题标签库

素材库已更新！下次创作时，我会自动检索这些素材。

## 注意事项

1. **文件夹命名**：必须使用日期前缀，方便按时间排序
2. **内容完整性**：尽量保存完整信息，包括数据表现
3. **提取质量**：只提取真正有价值、可复用的元素
4. **分类清晰**：金句、标题、标签要分类存储
5. **图片处理**：如果用户提供图片路径或图片，要保存到对应位置

## 关联能力

- **检索素材**：记录的素材会被检索能力使用
- **生成标题**：标题库会被生成标题能力使用
- **生成文案**：金句库会被生成文案能力使用
- **推荐话题**：话题标签库会被推荐话题能力使用
